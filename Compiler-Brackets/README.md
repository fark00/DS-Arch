# Let's Check brackets like a Compiler! 

## In this program, the correctness of opening and closing the brackets is checked.

### **remark**

Types of brackets supported in this program:
* ( )
* \[ \]
*  { }
*  (* \*) (Incorrect: (\*))
*  < >


***Correct input:*** 

- (*<{({})[]{[()()]}}>*)

- (){}(())

***Incorrect input:***

- ()(

- ()(){<}>
